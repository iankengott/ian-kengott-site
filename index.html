MANTiS / X-ray Spectromicroscopy

Scientific tooling, packaging, and analysis workflows

This repository started from my Nix package for MANTiS, a multivariate x-ray spectromicroscopy analysis tool. My spectroscopy notes cover the MANTiS workflow: loading hyperspectral data, preprocessing, PCA/SVD, clustering, spectral maps, NNMA, peak fitting, and tomography context.

Principles

Build, verify, simplify.

  • Own the stack. Prefer tools that can be inspected, moved, and repaired.
  • Make it useful early. A rough working system teaches more than a polished guess.
  • Stay grounded. Research pages should distinguish confirmed work from context and avoid inflated claims.
  • Protect the private parts. Public work should show taste and direction without leaking operational details.
  • Automate after understanding. The best automation starts with knowing where the friction really is.
